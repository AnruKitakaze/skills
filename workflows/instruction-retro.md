# Instruction Retro Workflow

Use after a process failure, repeated correction, or user intervention caused by unclear agent instructions.

1. Orchestrator may suggest this workflow but must not run it automatically.
2. Instruction Maintainer identifies the narrow instruction gap.
3. It proposes changes to agents, skills, workflows, checklists, or knowledge.
4. User approves or rejects the proposal.
5. Only then are files edited.

Avoid turning one-off preferences into global rules.
