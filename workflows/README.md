# Workflows

Reusable orchestration patterns. Workflows describe when to call agents, what context to pass, and when to stop for the user.

- **[taskforge](./taskforge.md)** - default flow for TaskForge issues with a DoD.
- **[feature](./feature.md)** - feature implementation.
- **[bugfix](./bugfix.md)** - bug or regression work.
- **[security-review](./security-review.md)** - standalone security review.
- **[instruction-retro](./instruction-retro.md)** - improve the agent system after process failures.
