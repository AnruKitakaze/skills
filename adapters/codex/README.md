# Codex Adapter

Codex already works well for this setup, so keep its integration conservative.

Recommended use:

- Keep `AGENTS.md` short.
- Do not auto-load all role prompts.
- For complex tasks, manually ask Codex to consult a specific workflow or role file.
- Prefer OpenCode when you want automatic specialist agents.

Useful entry points:

- `agents/orchestrator/AGENT.md`
- `workflows/taskforge.md`
- `agents/architect/AGENT.md`
- `agents/security-reviewer/AGENT.md`
