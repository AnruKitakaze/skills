# Adapters

Thin integrations for specific AI CLIs. The source of truth stays in top-level `agents/`, `workflows/`, `checklists/`, `knowledge/`, and `skills/`.

- **[opencode](./opencode/README.md)** - first-class adapter that generates `.opencode/opencode.json`.
- **[codex](./codex/README.md)** - conservative manual guidance to avoid automatic context growth.
